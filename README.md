# excleOnlie
	基于django开发的在线填写excel资料的网站

## 2019/2/1[最近更新]

	目前只实现了核心的基础功能，这个坑不知道什么时候填上:
		已经实现了注册的功能，注册后还不能直接进行登录，还需要通过超级管理员在后台添加到网站成员才可以登录到主页面。
		用户可以在线填写管理员发布的工作表，并且通过字典的格式保存数据到数据库。
		功能尚且简陋，如果确定有需求会继续进行更新完善。
